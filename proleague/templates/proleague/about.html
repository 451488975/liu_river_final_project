About

The HHO E-sports League Tournament Information System is a Web-based application for the HEL tournament database. Here, you can check out all the information about previous tournaments we hosted before. From games, teams, players to single match details, this is the ultimate E-sport Hub for what you want. This application is implemented using Django.
